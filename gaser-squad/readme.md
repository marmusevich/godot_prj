# Global / Universe {#global-universe}
### [ru] 

Глобальный режим игры.

Здесь игрок управляет материнским кораблём:
- перелёты между звёздными системами  
- перемещения между планетами  
- исследование галактики  

Возможности:
- космические сражения  
- абордаж (переход в тактический бой)  

Референс: *Космические рейнджеры*

# Planets / Space Objects {#planets-space-objects}
### [ru] 

Режим взаимодействия с планетами и космическими объектами.

Здесь доступны:
- правительственные взаимодействия  
- квесты  
- боевые события  
- исследование поверхности  
- специальные события  

Формат представления:
- либо карта планеты с иконками заданий  
- либо список доступных событий (требует уточнения)  

Один из входов в тактический режим.

Референс: система взаимодействия как в *Космических рейнджерах*.

Возможно в будущем:
- разделение режимов для планет и космических станций.

# Tactical Battle {#tactical-battle}
### [ru] 

Тактический пошаговый бой.

Особенности:

- автогенирация карт в некоторых случаях

Референсы:
- *X-COM* (миссии)  
- *Laser Squad*
